# [PBI] 仮会員証を発行し、表示できる

## 概要（仕様サマリー）
ユーザーが選択した店舗で利用できる仮会員証（QRコード）を発行する機能を提供する。
仮会員証は30分間有効で、店舗スタッフにQRコードを提示することで本会員証にアクティベートできる。
POS様システムと連携し、仮会員情報を登録する。

## 背景 / 目的
- 店舗での本会員証アクティベートのため、事前に仮会員証を発行する必要があるため
- QRコードにより店舗スタッフが簡単に会員情報を取得できるようにするため
- 有効期限を設けることで、未使用の仮会員証を自動的に無効化するため

## 仕様

### 仮会員証の仕様
| 項目 | 内容 |
|------|------|
| 有効期限 | 発行から30分間 |
| QRコード | POS様システムから発行されたコードを表示 |
| ステータス | temporary（仮会員） |
| TTL | 30分後に自動削除 |

### 処理フロー
1. 店舗詳細画面で「仮会員証を発行」ボタンを押下
2. バックエンドからPOS Gatewayに仮会員登録リクエスト
3. POS GatewayからQRコードと仮会員IDを取得
4. DynamoDBに仮会員証レコードを作成
5. ユーザーにQRコードを表示

### 画面遷移
- 店舗詳細画面 → 仮会員証発行ボタン → 仮会員証表示画面

## UI（Figma）
- 仮会員証表示画面（QRコード、有効期限カウントダウン表示）
