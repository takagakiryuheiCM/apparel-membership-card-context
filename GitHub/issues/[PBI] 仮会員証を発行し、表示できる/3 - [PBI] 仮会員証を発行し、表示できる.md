# [PBI] 仮会員証を発行し、表示できる

## Issue情報
- **ステータス**: ✅ Done
- **ラベル**: product backlog
- **マイルストーン**: Sprint 2
- **担当者**: @takahashi

## 概要

ユーザーとして、選択した店舗で利用できる仮会員証を発行したい。
そうすることで、店舗でアクティベートして本会員になることができる。

## 受け入れ条件

- [ ] 店舗詳細画面から仮会員証発行ボタンを押せる
- [ ] 仮会員証（QRコード）が表示される
- [ ] 有効期限（30分）が表示される
- [ ] 有効期限切れの場合、再発行ボタンが表示される
- [ ] POS様システムに仮会員情報が登録される
