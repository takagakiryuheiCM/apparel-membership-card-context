# [PBI] 退会できる

## Issue情報
- **ステータス**: ⏳ Todo
- **ラベル**: product backlog
- **マイルストーン**: Sprint 3
- **担当者**: 未割り当て

## 概要

ユーザーとして、会員証サービスから退会したい。
そうすることで、個人情報を削除してサービスの利用を終了できる。

## 受け入れ条件

- [ ] マイページから退会ボタンを押せる
- [ ] 退会確認画面が表示される
- [ ] 確認後、すべての会員証が無効化される
- [ ] ユーザー情報が論理削除される
- [ ] 退会完了画面が表示される

## 技術的な詳細

### 関連ドキュメント
- [DB設計](../../wiki/3.設計/DB設計.md) - 論理削除の仕様

### API
- `DELETE /api/users/me` - 退会

### 画面
- 退会確認画面（/withdraw/confirm）
- 退会完了画面（/withdraw/complete）

## 備考
- 退会後30日間はデータを保持（問い合わせ対応用）
- 30日後に物理削除するバッチは第二フェーズで検討
