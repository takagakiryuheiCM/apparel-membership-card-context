# [PBI] 店舗一覧が表示でき、検索できる

## 概要（仕様サマリー）
ユーザーが会員証を発行したい店舗を探すため、対応店舗の一覧を表示する機能を提供する。
店舗名やエリアでの検索・絞り込みができ、店舗をタップすると詳細画面に遷移する。

## 背景 / 目的
- ユーザーが仮会員証を発行する店舗を選択するために、店舗の一覧が必要なため
- 店舗数が多くなることを想定し、検索・絞り込み機能でユーザビリティを向上させるため
- 店舗の基本情報（名前、エリア、住所）を一覧で確認できるようにするため

## 仕様

### 基本仕様
| 項目名 | 表示 | 説明 |
|--------|:----:|------|
| 店舗名 | ○ | 店舗の正式名称 |
| エリア名 | ○ | 渋谷エリア、新宿エリア等 |
| 住所 | ○ | 都道府県+市区町村+住所 |

### 検索・絞り込み
| 機能 | 説明 |
|------|------|
| キーワード検索 | 店舗名での部分一致検索 |
| エリア絞り込み | エリア名での絞り込み |

### 処理フロー
1. トップページから店舗一覧画面を開く
2. 店舗一覧を表示（初期表示は全店舗）
3. 必要に応じて検索・絞り込み
4. 店舗をタップして詳細画面へ遷移

### 画面遷移
- 店舗タップ → 店舗詳細画面へ遷移
