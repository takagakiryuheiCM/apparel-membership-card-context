# [PBI] 店舗一覧が表示でき、検索できる

## 概要

ユーザーとして、対応している店舗の一覧を見たい。
そうすることで、会員証を発行したい店舗を探すことができる。

## 受け入れ条件

- [ ] 店舗一覧画面が表示される
- [ ] 店舗名、エリア、住所が表示される
- [ ] 店舗名で検索できる
- [ ] エリアで絞り込みができる
- [ ] 店舗をタップすると詳細画面に遷移する
