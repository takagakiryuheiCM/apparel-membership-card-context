# [PBI] 会員情報が登録できる

## Issue情報
- **ステータス**: ✅ Done

- **マイルストーン**: Sprint 1
- **担当者**: @takahashi

## 概要

ユーザーとして、LINEミニアプリ上で会員情報を登録したい。
そうすることで、各店舗で会員証を発行できるようになる。

## 受け入れ条件

- [ ] LINEログイン後、会員情報入力画面が表示される
- [ ] 必須項目（氏名、フリガナ、生年月日、性別、住所、電話番号）が入力できる
- [ ] 入力値のバリデーションが行われ、エラーメッセージが表示される
- [ ] 登録完了後、トップ画面に遷移する
- [ ] 登録した情報がDynamoDBに保存される
