## 画面仕様

### 入力項目
| 項目 | 必須 | 入力形式 |
|------|------|----------|
| 姓 | ○ | テキスト（64文字以下） |
| 名 | ○ | テキスト（64文字以下） |
| せい | ○ | ひらがな（64文字以下） |
| めい | ○ | ひらがな（64文字以下） |
| 生年月日 | ○ | 日付選択 |
| 性別 | ○ | ラジオボタン |
| 郵便番号 | ○ | 数字（7桁） |
| 都道府県 | ○ | セレクトボックス |
| 市区町村 | ○ | テキスト（30文字以下） |
| それ以降の住所 | ○ | テキスト（40文字以下） |
| 建物名 | - | テキスト（40文字以下） |
| 電話番号 | ○ | 数字（10-11桁） |

### バリデーション
- 必須項目の入力チェック
- フリガナはひらがなのみ
- 電話番号は0始まり
- 郵便番号は数字7桁
