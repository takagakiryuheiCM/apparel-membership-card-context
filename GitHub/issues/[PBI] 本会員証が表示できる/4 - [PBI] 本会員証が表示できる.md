# [PBI] 本会員証が表示できる

## Issue情報
- **ステータス**: 🔄 In Progress
- **ラベル**: product backlog
- **マイルストーン**: Sprint 3
- **担当者**: @suzuki

## 概要

ユーザーとして、アクティベート済みの本会員証を表示したい。
そうすることで、店舗でポイントを貯めたり使ったりできる。

## 受け入れ条件

- [ ] 本会員証（バーコード/QRコード）が表示される
- [ ] 会員番号が表示される
- [ ] ポイント残高がリアルタイムで表示される
- [ ] ポイント有効期限が表示される
- [ ] 店舗情報（店舗名、住所）が表示される

## 技術的な詳細

### 関連ドキュメント

### API
- `GET /api/membership-cards/{cardId}` - 会員証詳細取得（ポイント含む）

### 画面
- 本会員証表示画面（/membership-card/{cardId}）

## 備考
- ポイント取得はPOS様APIをリアルタイム呼び出し
- キャッシュは行わず、常に最新のポイントを表示
