# 工数見積もり

## 概要

アパレル会員証LINEミニアプリ開発の工数見積もりです。

## 見積もりサマリー

| フェーズ | 工数（人日） | 期間 |
|----------|-------------|------|
| 要件定義 | 15人日 | 2025/11/05 - 2025/12/06 |
| 設計 | 10人日 | 2025/12/09 - 2025/12/20 |
| 開発 | 40人日 | 2026/01/06 - 2026/02/21 |
| テスト | 10人日 | 2026/02/24 - 2026/03/13 |
| リリース準備 | 5人日 | 2026/03/16 - 2026/03/25 |
| **合計** | **80人日** | |

## 前提条件

- 開発メンバー: 2名（BE 1名、FE 1名）
- 1人日 = 8時間
- 祝日・年末年始休暇を考慮

## 注意事項

- 本見積もりは2025年11月時点のものです
- 要件変更が発生した場合は再見積もりが必要です
